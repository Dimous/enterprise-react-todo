# enterprise-react-todo

A demonstration of enterprise architecture for a React.js application. _I'm convinced that applications must have architecture. Unfortunately, bootcamps have churned out too many code monkeys who consider architecture overengineering. Architecture isn’t about having a state manager or a scaffold. Rarely, though, is even that present. I wouldn't care if I didn't have to maintain their crap. Unfortunately, there are no companies with an engineering culture in the local market._

## Setup

Install the dependencies:

```bash
npm install
```

## Get started

Start the dev server, and the app will be available at [http://localhost:3000](http://localhost:3000).

```bash
npm run dev
```

Build the app for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
